# `creditopia`

## pre
- install dfx
- install npm
- install cargo `curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh`
- install wasm `rustup target add wasm32-unknown-unknown`
- `dfx start`

## local installation
1. clone repo `git clone `
2. go to directory `cd creditopia`
3. `npm install`
4. `cargo build`
5. `dfx deploy`
